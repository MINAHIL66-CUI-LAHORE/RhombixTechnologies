## Project Description
This project is a music recommendation system inspired by Spotify’s recommendation approach. It uses machine learning to predict whether a user will listen to a song again within a given time period.

The dataset contains user-song interaction history along with timestamps. The target variable indicates whether a user replayed a song within one month (1 = repeated listen, 0 = not repeated). The system learns patterns from user behavior to generate personalized recommendations.

---

## Objective
- Analyze user listening behavior  
- Identify patterns in song replay frequency  
- Predict likelihood of song replay  
- Build a machine learning-based recommendation system  
- Provide a simple interactive web application using Streamlit
